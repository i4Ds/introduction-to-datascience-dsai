## Von Daten zu Insights: Data Science & AI

Für diverse Workshops zum Studiengang Data Science und AI werden in diesem Repository Dateien zum Programm abgelegt:
- Im Ordner "orange_data_mining" finden sich Dateien für die Software Orange, welche eine interaktive Transformation und Visualisierung der Daten bietet, und wo bereits erste Modells trainiert werden können. Grundlage ist das Data-Set "Krankenkasse" aus einer früheren EDA Challenge.
- Im Ordner "python" findet sich eine Vorlage für die Analyse mithilfe von Python. Grundlage ist das Data-Set "Krankenkasse" aus einer früheren EDA Challenge. Mithilfe von Binder kann diese Notebook interaktiv von den Workshopteilnehmer genutzt und bearbeitet werden.
